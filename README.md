# User Dashboard Explorer

A responsive user dashboard built with HTML, CSS, and JavaScript that fetches user data from the Random User API. The app allows users to search, filter, and view detailed information through an interactive interface.

## 🚀 Live Demo

https://jprobussr.github.io/dashboard-user-explorer/

## 📌 Features

- Fetch and display user data from an API
- Dynamic user cards with images and details
- Live search filtering by name and email
- Selected user detail panel
- Empty state handling
- Responsive design (mobile-first)

## 🛠️ Tech Stack

- HTML
- CSS (Design tokens + responsive layout)
- JavaScript (ES6+)
- Random User API

## 🧠 What I Learned

- Working with APIs using `fetch` and async/await
- Managing application state (users, filtered users, selected user)
- Rendering dynamic UI with JavaScript
- Building interactive components (search, detail view)
- Creating responsive layouts with CSS Grid and Flexbox
